Para comprimir un fichero en formato .zip, lo haremos de esta forma:
```bash
zip -r archivos.zip documento.txt
```
![[Pasted image 20230323032724.png]]
Si queremoms descomprimir el archivo usaremos el comando unzip:
```bash
unzip archivos.zip
```
